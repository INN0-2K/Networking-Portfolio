# TechCorp Two-Site Network
 
A simulated enterprise network built in Cisco Packet Tracer demonstrating a realistic two-site topology for a small business. This project covers VLANs, inter-VLAN routing, OSPF, DHCP, NAT, and ACLs — core skills from the Cisco CCNA curriculum applied in a practical scenario.
 
---
 
## Project Overview
 
TechCorp is a small company with a **headquarters (HQ)** and a **branch office**, connected over a simulated WAN link. The goal was to design and configure a network that a real business could plausibly use — with proper segmentation, redundancy, centralized services, and basic security.
